Safely is a Community-driven Research project

Smartphones know a LOT about us. They know our work-life patterns, where we parked,
how many steps we took, who we met and more.

Safely is an attempt to answer the question,
"Can our smartphones become an always-on assistant for personal safety and well-being ?

Safely uses the SensorHub to determine
anamolous behavior patterns.

SensorHub can run a Deep Neural Network for background detection of anomalous patterns. SensorHub's low power consumption allow it to run all the time with negligible battery drain.

Safely integrates with 3rd party
systems such as campus security.

Depending on the location, Safely can integrate with various security systems for automated alerts. For example, when at home Safely integrates with a community/neighborhood network. When at work, it integrates with the campus security system.

Safely is a platform where
developers can build "modules".

"Modules" allow for context specific behavior. For example, a "hiking" module can create a group of people on the hike. This can include real-time location tracking, ensuring nobody is lost and integration with the Park Rangers system.
