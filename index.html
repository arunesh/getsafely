Safely is a Community-driven Research project

Smartphones know a LOT about us. They know our work-life patterns, where we parked,
how many steps we took, who we met and more.

Safely is an attempt to answer the question,
"Can our smartphones become an always-on assistant for personal safety and well-being ?"

Safely uses Sensors + AI to
detect anamolous events.

SensorHub can run a Deep neural networks for background detection of anomalous patterns. Such Smart algorithms designed by top developers can help identify anamalous behavior patterns automatically. SensorHub's low power consumption allow inferencing to run all the time with negligible battery drain.

Safely integrates with 3rd party
systems such as campus security.

Depending on the location, Safely can integrate with various security systems for automated alerts. For example, when at home Safely integrates with a community/neighborhood network. When at work, it integrates with the campus security system.

Safely is a platform where
developers can build "modules".

"Modules" allow for context specific behavior. For example, a "hiking" module functions with a group of people who are on the hike. This can include real-time location tracking, ensuring nobody is lost and integration with the Park Rangers system.

Safely as a platform for personal safety and well being.

Safely is primarily a voice-based AI assistant on the phone. Safely is always aware of the user's location and context. It looks for any deviations in regular patterns and combines this with inferencing from AI/Deep learning algorithms to make a judgement.
Below are some examples of scenarios where Safely as a platform can really add value.

  • Safely at Home

    Safely can monitor unusual activites at home including abnormal patterns for seniors or any family member. This can save valuable time when calling for help. Developers can create customizations for family members and care environments.

  • Safely at Work

    Safely can monitor deviations from work patterns and notify corporate security departments. At an aggregate level, Safely can monitor employee security at scale.

  • Safely for sports

    Safely modules can monitor user activity during sports and notify when situations arise where the user is unable to ask for help.

  • Safely for events

    Special events, such as bus tours, parties, special events etc can have modules that cater to their unique characteristics. Safely can integrate with the event organizer's system for notifications and alerts.

  • Safely for hikes

    As an example, in the situation of a group-hike or exercise-groups, Safely can monitor the relative location of the group of people and notify the Park authority if something unusual happens.

  • Safely for ride-shares

    Safely can monitor unusual activity when taking a taxi for example to reduce instances of harm. This can help reduce incidents during ride-shares.
